# StudentDispatch

## Build APK on GitHub
This repo includes a GitHub Actions workflow that builds a **Debug APK** on each push.

1. Go to **Actions** tab.
2. Open **Build Debug APK** run.
3. Download the artifact **app-debug-apk**.

The output file is: `app/build/outputs/apk/debug/app-debug.apk`.

## Build APK on GitHub (GitHub Actions)

This repo includes a GitHub Actions workflow that builds a **debug APK** on every push to `main`/`master`.

- Go to **Actions** tab → **Build Debug APK** → open the latest run → download the **app-debug-apk** artifact.
